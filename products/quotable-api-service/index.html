产品介绍

言屿 API（quotable.cn）是一个专业的句子 API 服务平台，致力于收集和整理优质的名言警句、励志语录、文学摘抄等文本内容。平台提供稳定可靠的 API 接口，支持每日随机获取一条或多条名言警句，支持按类别搜索指定语句库，支持数据导出和图片生成等功能。每天免费调用 1000 次，为开发者、内容创作者和应用开发者提供便捷的句子数据服务。

主要特性

  • 每日随机名言警句 API
  • 支持批量获取多条句子
  • 按类别搜索指定语句库
  • 支持多种数据格式返回（JSON、XML、TXT）
  • 数据导出功能（Excel、CSV、PDF）
  • 句子图片生成服务
  • RESTful API 接口设计
  • 详细的 API 文档和示例代码
  • 多语言支持（中文、英文等）
  • 高频调用限制和防护机制
  • 数据统计和分析功能
  • SDK和代码示例提供
